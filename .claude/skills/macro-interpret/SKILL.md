# 경제지표 해석 프레임

## 3축 분해
- **성장**: 서비스 PMI(S&P Global vs ISM 양 갈래 비교), 주택판매 → 모멘텀 피크아웃·소비 여력
- **인플레이션**: 서비스 PMI 둔화 → 인건비 압력 완화 → 연준 우호 여부
- **노동**: JOLTS → "balanced labor market" 내러티브 부합 여부, 급랭 vs 연착륙

## 종합 → 시장 매핑
- "성장 둔화 + 인플레 완화 + 주택 회복" 조합이면 **골디락스**로 명명
- 골디락스 4요소 체크: 주택강세(소비), 서비스 50+(확장지속), 노동 미냉각, 인플레 압력↓
- 지수 상승률 순위(RUSSELL > NASDAQ > S&P > DOW)에서 **금리 베타 위계**를 읽어 인하 기대 강도 추론
- 같은 방향 신호(예: ISM 서비스 둔화 + JOLTS 쿨링 = 서비스 인건비 디스인플레)는 묶어서 한 줄로 해석

## 출력 포맷
- 표는 발표치/예상/이전/방향 4컬럼 유지
- 데이터 표 + `→` 분석 한 단락 (블로그 일일 리뷰와 동일 흐름)
