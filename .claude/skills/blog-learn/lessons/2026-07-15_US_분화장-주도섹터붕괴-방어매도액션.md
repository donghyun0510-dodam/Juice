## 분류된 패턴

draft와 발행본의 1~4번(뉴스·매크로·지수·섹터)은 사실상 동일(네이버 에디터의 표/볼드 → 줄바꿈 노이즈만). **의미 있는 편집은 전부 5.종합 판단과 매크로 종합 라인에 집중**됨.

### 삭제
- **매크로 종합 라인 꼬리 "VIX는 진정됐으나 엔 162엔대와 함께 잔존 불안" 삭제.** VIX·엔은 각 매크로 항목에 이미 있어 종합 라인에서 반복 → 중복 컷. (④ 가독성 / 중복 배치)
- **종합 판단의 독립 bullet "잔존 위험은 10Y·30Y 높은 장기금리 + 엔캐리 부담" 통삭.** 매크로 종합 라인과 동일 내용이라 종합 판단에서 중복 → 제거. (④ / 중복 배치)

### 어조 (핵심)
- **무매매 "유지·보류" 마무리 → 실제 % 축소 액션으로 격상.** draft는 "방어 스탠스 유지 + 국내 반도체 Long Sign 재점등까지 재증액 보류"(무매매)로 닫았으나, 발행본은 **"반도체 비중 추가 축소 필요 — 삼전닉스는 편출에 가까운 축소, 주식 35% → 30%"**로 실제 매도 액션을 % 하드값으로 단정. 이날은 보유 주도섹터(메모리·AI H/W)가 Sell·Short 다발로 무너진 **본인 방어 액션일**이었는데 draft가 이를 "유지"로 과소평가함. (③ 진정성 — 실제 취한 액션 반영 / §1 주도주 Sell 다발 방어 행)

### 사실추가
- **빅테크 상승의 질 판정 추가**: "빅테크에서 상승이 나타났지만 아직 랠리로 보기 부족(상승 추세는 AAPL 밖에 없기 때문)". 지수가 올라도 **상승추세를 회복한 종목이 소수면 랠리로 규정하지 않음**. (① 전문성 / ⑥ 신호 위계)
- **재증액(국면 전환) 조건 구체화**: "AI H/W가 안정 조절 레벨을 이탈 + 빅테크 전반 Long Sign 발생 → 주도 섹터 변화 → 포트 수정·비중 확대". draft의 막연한 "Long Sign 재점등"을 두 조건으로 명시.

### 표현
- 표 아래 → 요약 "매파 증언" → "매파적 발언"(경미).
- 1.뉴스 표 정치 이벤트 칸 "워시 Fed 의장 증언" → "Fed 연설 / - 워시 의장"으로 압축.
