## 분류된 패턴

(6/15 세션 — 미-이란 종전 MOU 타결發 유가 급락·위험선호 회복, KOSPI +5.20% / KOSDAQ +0.48% 대형주 쏠림 강세장. 외국인·기관 코스피 쌍끌이 매수.)

### 삭제

- **1) 금리 데이터 줄의 "단기물 하락" 삭제** — draft "국채 3년물 3.747% (-1.91%). 단기물 하락, 채권 강세." → 발행 ". 채권 강세." 등락률에 이미 방향(-)이 있으면 "단기물 하락"은 중복. 매크로 항목은 데이터 + 짧은 한 마디만.
- **종합 판단 마지막 워치 `→` 라인 통째 삭제** — draft 3번째 bullet 뒤 "→ 외국인의 코스닥 매도·중소형 소외가 지속되는지, 대형주 쏠림이 폭을 넓히는지 확인하며 환원 속도를 조절." 전체 삭제. 임박 이벤트가 없고 본인 액션이 이미 확정(증액)되면, 모니터링 워치를 덧붙이지 말고 **액션 단정으로 글을 닫음**.
- **종합 판단 주도주 목록에서 "전력기기" 제외** — draft "주도주(반도체·금융·전력기기)" → 발행 "주도주(반도체·금융)". 전력기기는 섹터 본문에선 다루되(고점 대비 조정폭 잔존), **본인이 실제 증액하는 주도주 구성에선 뺌**. 섹터 강세 ≠ 주도주 편입 대상.

### 어조

- **환율 인사이트 "우호적"(평가) → "순매수 시작"(사실 단정)** — draft "원화 소폭 강세 — 외국인 코스피 순매수에 우호적" → 발행 "원화 소폭 강세 — 외국인 코스피 순매수 시작." 평가형 형용어보다 실제 일어난 사실(순매수 시작)을 단정.
- **전자부품 → 인사이트: 단순 강세 묘사 → 추세 맥락 + 한계 단서** — draft "전장·카메라모듈 중심 두 자릿수 큰 폭 강세." → 발행 "최근 급격한 조정에서 벗어나는 급등이나 아직 재랠리 기대하기에는 무리." 등락률 재진술(두 자릿수) 대신 **"조정 탈피하되 재랠리는 무리"** 추세 위치·한계로 격상.

### 표현

- **원자재 유가 인과 = 짧게 (표에 MOU가 있으면 원자재는 압축)** — draft "**미-이란 종전 MOU 타결**로 호르무즈 해협 개방 기대가 커지며 유가 급락." → 발행 "중동 종전 기대에 유가 급락." 시장 뉴스 표에 종전 MOU 행이 이미 있으므로 원자재 코멘트는 **데이터 + 짧은 인과 한 줄**로. (표 행 + 매크로 인과 전담의 분업 — 인과를 양쪽에 길게 중복 X.)
- **지수 인사이트 수식 압축** — draft "상승이 대형주에 **극단적으로 집중된 비대칭 강세**." → 발행 "상승이 **대형주에 집중**." "극단적으로"·"비대칭 강세" 강조 수식을 덜고 사실만.

### 구조

- **섹터 주도 순서 = 그날 실제 주도 축을 1순위로 (반도체를 금융 앞으로)** — draft 순서 금융 → 반도체·소부장 → 전자부품 → 전력기기. 발행 순서 **반도체·소부장 → 전자부품 → 금융 → 전력기기**. Long Sign 다발은 금융이지만, 그날 강세를 *촉발한 주도 축*(금요일 미국 반도체 급등 + 종전 뉴스)이 반도체라 **반도체·소부장을 맨 앞으로**. 신호 다발 섹터 ≠ 자동 1순위 — 그날 시장을 끈 축이 1순위.
- **종합 판단 마지막 = 모호한 "유지·환원 고려" → 구체 실제 매매 액션 단정** — draft "주도주(반도체·금융·전력기기) 중심으로 비중을 유지함. 누적 현금(20% 수준)은 강세가 이어질 경우 단계적 환원을 고려." → 발행 "**주도주(반도체·금융) 중심으로 5%를 증액함(현금 20% → 15%).**" "유지"·"고려" 같은 모호어 대신 **실제로 취한 액션(증액 폭·현금 비중 변화 수치)을 확정**해 닫음. (강세 추세 유효 판단 → 행동까지 단정.)

### 사실추가

- **주변국 데이터에 인트라데이 사실 보강** — "니케이225 +5.33%" → "니케이225 +5.33%**(고점 갱신)**".
- **반도체 → 인사이트에 선행 트리거 명시** — draft "대형주와 소부장이 동반 강세" → 발행 "**금요일 미국 반도체 급등과 종전 뉴스 영향으로** 반도체 대형주와 소부장이 동반 강세". (알려진 선행 트리거는 인사이트에 직접 박음.)
- **전력기기 → 인사이트에 글로벌 피어 연결** — 발행본 추가: "글로벌 증시에서 피어 그룹인 **GEV, VRT가 단기 하락 추세를 반전시킬 만한 상승**을 보여준 것과 유사한 점은 AI 밸류체인으로 반도체와 함께 다시 묶일 수 있다는 낙관적 전망을 하게 함." (국내 전력기기를 미국 피어(GEV·VRT) 흐름과 묶어 AI 밸류체인 재편입 가능성으로 해석.)
- **종합 bullet1 인과 구체화** — draft "원/달러 강세가 외국인 순매수와 맞물려" → 발행 "**무역 흑자 뉴스와 중동 종전 소식으로 달러 상승이 주춤한 부분**이 외국인 순매수와 맞물려". 환율 단일 요인 → (무역흑자 + 종전 → 달러 주춤) 인과사슬로.
