## 분류된 패턴

7/1 미장(반도체 차익실현 로테이션·빅테크/S·W/은행 강세·워시 매파에 금리 상승, 사용자 이미지 논조를 종합 판단에 반영). draft 골격·데이터는 통과, **종합 판단 마무리의 다단계 해석 프레임을 방어적 단일 액션으로 압축**한 것이 핵심.

### 삭제
- **종합 판단 2번 bullet의 "관점 수정 불필요·관망으로 충분" 낙관 tail 삭제**. draft "…주도주 추세의 최악의 훼손은 나타나지 않음 **→ 투자 관점을 서둘러 수정할 요소는 아직 보이지 않으며, 상승 흐름 대응·관망으로 충분.**" → 발행은 "…최악의 훼손은 나타나지 않음"에서 끊음. 이미지(사용자 원문)가 "관점 서둘러 수정할 요소 안 보임·관망"이라 해도, **본인 실포지션이 걸린 종합 판단은 낙관 관망으로 닫지 않음**(뒤의 방어 액션과 톤 상충 방지).
- **종합 판단 마지막 →의 삼성전자→SK하이닉스 다단계 시나리오 트리 통째 삭제 → 단일 조건부 축소 액션으로 압축**. draft "미 반도체 조정은 전일 삼성전자에 이미 반영됐으므로 금일 충격 제한적이어야 함 / 삼성전자 그 이상 이탈 시 개별 문제 / SK하이닉스 확산 시 메모리 부진 재점화 / AI H/W 관점 추가 확인" (4단계 시나리오) → 발행 "**전일에도 국내 반도체는 하락 폭이 적지 않았으므로 금일까지 하락이 심해진다면 비중 축소가 진행될 것.**" 다단계 해석 프레임은 draft 사고 과정으론 OK지만 발행 결론부는 *오늘 본인이 취할 단일 액션*으로 압축.
- **S/W → 인사이트에서 "PANW Long Sign" 삭제**. draft "PANW Long Sign, PLTR·NOW는 Short Cover Sign으로…" → 발행 "PLTR·NOW는 Short Cover Sign으로 단기 하락 극복". 인사이트는 그날 로테이션의 대표 신호(Short Cover 다발)만 남기고 종목 나열성 신호 언급 축소.
- **시장 뉴스 표 이벤트 칸 "(Warsh)" 인물명 삭제** → "연준". 이벤트 칸은 짧은 토픽/기관명만(인물명은 내용 칸 "캐빈 워시 의장"에 이미 있음).

### 어조
- **이미지(사용자 원문)의 "관망" 낙관 → 본인 실포지션은 "방어적 조건부 축소"로 전환**. 이미지 논조는 "관점 수정 불필요·그냥 관망"이었으나, 발행 결론은 "**전일에도 국내 반도체 하락 폭이 적지 않았고, 금일까지 심해지면 비중 축소**"로 방어 스탠스. 즉 시황 해석(주도주 추세 미훼손)은 이미지 톤을 따르되, **본인 매매 액션은 한 단계 더 보수적으로** 닫음. ("충격은 제한적이어야 함"=기대·관망 → "하락 심해지면 축소"=방어 액션.)
- **"~변화가 아님"(부정 단정) → "~변화는 없었음"(사실 서술)**. 종합 "AI 펀더멘탈의 변화가 아님" → "AI 펀더멘탈의 변화는 없었음". 같은 뜻이나 담백한 사실 서술체.

### 표현
- **로테이션 지배 요인어 "가격 요인" → "차익 실현"**. draft가 이미지 원문의 표현("가격 요인이 지배한 로테이션")을 그대로 썼으나 발행은 "**'차익 실현'이 지배한 로테이션**". 블로그 표준 어휘는 "차익 실현"(반도체 상반기 +80% 급등 되돌림 맥락).
- **같은 문장 내 "차익실현" 중복 회피** — 앞 절 주어 교체. "**반도체 차익실현이** 지수를 눌렀으나 … '차익 실현'이 지배한 로테이션" → "**반도체의 대거 하락이** 지수를 눌렀으나 … '차익 실현'이 지배한 로테이션". 뒤에 "차익 실현" 핵심어가 오면 앞 주어는 "대거 하락"으로 바꿔 중복 제거.
- **신호명 "Short Sign" → 서술적 상태 규정 "long 전환 실패"** [전력/원전]. draft "VRT Short Sign" → 발행 "**VRT long 전환 실패**". 추세 약세 섹터에서 하락 종목의 신호를 딱딱한 "Short Sign"보다 *그 종목이 기대됐던 방향(long 전환)에 실패*했다는 서술로 — TLN Sell Sign·CEG 52주 신저가와 함께 "AI 인프라 반전 실패" 서사에 정합.

### 구조
- **종합 판단 마무리 = 다단계 해석 프레임(시나리오 트리)이 아니라 단일 조건부 액션**. 미장 리뷰라도 본인 실포지션(국내 반도체) 관점 cross-market 마무리는 유지하되, "A이면 B·C이면 D·E는 추가 확인" 식 분기 나열 대신 "**전일 맥락 + 금일 조건 → 단일 액션(축소)**" 한 문장. (line 202 "워치 다변수 나열 대신 단일 임박 이벤트"의 액션 버전 — 결론부는 시나리오 백화점이 아니라 오늘의 한 수.)

### 사실추가
- **본인 실포지션 액션 근거에 "전일 하락 맥락" 추가**. 발행 "**전일에도 국내 반도체는 하락 폭이 적지 않았으므로** 금일까지 하락이 심해진다면 비중 축소". 미 반도체 조정의 국내 전이를 *전일 이미 시작된 하락의 연장*으로 규정해, 금일 추가 하락 시 축소라는 조건부 액션의 근거로 삼음(사용자가 아는 국내 전일 흐름).
