## 분류된 패턴

### 삭제
- **원자재 데이터 줄 끝 해석어 삭제** — "유가 소폭 반락. → 이날만큼은 유가가 위험 요인에서 비껴남." 에서 `→ 이날만큼은 ...` 절을 통째 컷, "유가 소폭 반락."만 남김. 매크로 항목 코멘트는 데이터+짧은 사실 한 줄로 끊고, "위험 요인서 비껴남" 류 평가 해석어는 덜어냄. (기존 "데이터 한 줄 끝의 짧은 해석어 자제" 재확인·강화)
- **종합 판단 첫 bullet의 종목 수치 중복 삭제** — "...매도 사이드카 발동. 삼성전자 -6.40%·SK하이닉스 -9.92%로 **주도주가 동반 훼손**됨." 에서 삼성전자/하이닉스 수치 나열절 삭제. 1번·지수·섹터에서 이미 다룬 종목 수치는 종합 판단에서 재나열하지 않음.

### 어조
- **패닉장 현금 액션 = 20% 기준 (30% 아님)** — draft "현금 비중을 예수금 **30%** 수준까지" → 발행 "예수금 **20%** 수준까지". 매크로 더블 부담·사이드카 패닉장이라도 현금 권고 수치는 30%가 아니라 **20% 수준**이 사용자 선호. (SKILL.md 예시의 "30% 이상"은 과함 — 20%로 정정)

### 표현
- **시장 뉴스 표 분류 = "섹터", 이벤트 = "현상 명사(반도체 폭락)"** — draft 분류 "반도체"/이벤트 "브로드컴·마이크론" → 발행 분류 **"섹터"**/이벤트 **"반도체 폭락"**. 섹터 전반 모멘텀이면 분류 칸은 일반어 "섹터", 이벤트 칸은 고유명(브로드컴) 대신 **종목군+현상("반도체 폭락")**. 내용 칸엔 트리거→결과까지("...→ 미 반도체 약세 → **국내 반도체 폭락**") 박음.
- **지수 인사이트 비교 서술 압축** — "KOSPI200(-5.98%)이 **KOSPI보다 더 크게 빠지며** 대형주 집중 훼손" → "KOSPI200(-5.98%)이 **최대 하락으로** 대형주 집중 훼손". "A보다 더 크게 빠지며" 비교절을 "최대 하락으로" 명사형으로 압축.

### 구조
- **종합 판단 워치 마무리 = 나열형 `→ 워치:` X → 독립 bullet 단정형 액션 + 기회 시나리오** — draft "  → 워치: 외국인 매도 지속 여부, 원/달러 1,540원 돌파, 미 반도체주 익일 반등 여부." (하위 나열) → 발행은 **독립 bullet**으로 승격 후 단정형 문장 재작성: "원/달러 1,540원 돌파 여부와 미국 반도체 섹터 반등 여부를 통해 추가적으로 주식 비중 축소를 고려해야 함. 단, ... **반도체가 눌리면서 떠오르는 섹터를 편입**할 수 있음." 워치 항목 나열 대신 **모니터링→액션 단정형 문장**으로, 끝에 다음 국면 기회 한 줄 추가.
- **첫 bullet 하위 `→` 줄을 본문에 인라인 흡수** — draft 첫 bullet 아래 별도 줄 "  → 고금리·고환율 ... 배경." → 발행은 같은 bullet 문장 끝에 "...사이드카 발동. → 고금리·고환율 ... 배경."으로 인라인 결합(줄 분리 해제).

### 사실추가
- **수급 정확값 채움 + "개인만 매수" 압축 표기** — draft placeholder → 발행 "코스피: 외국인 [-3.5조], 기관 [-0.9조] / 개인만 매수" + "코스닥: 외국인 [-1,781억] / 기관 [+1,450억], 개인 [+337억]". **한 주체만 매수/매도면 수치 생략하고 "개인만 매수"로 압축 가능**(코스피처럼). 매도 주체 먼저 콤마로 묶고 슬래시 뒤 매수 주체. 코스닥처럼 3주체 다 유의미하면 정확값 전부 표기.
- **패닉장 마지막 = 기회 시나리오 한 줄 (전일 미장 섹터 로테이션 근거)** — "전일 미국 장에서는 주도주였던 반도체 섹터 위주 하락으로 다른 섹터는 상승 → 반도체 눌리며 떠오르는 섹터 편입 가능". 패닉 급락장이라도 종합 판단 마지막은 부정으로만 닫지 않고, **다음 국면의 섹터 로테이션 기회**를 단정형으로 한 줄 제시.
