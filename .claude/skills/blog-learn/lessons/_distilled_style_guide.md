# 도담아빠 블로그 — 발행본 학습 통합 스타일 가이드

`/blog-learn`으로 누적된 lesson들에서 추출한 정수 규칙. blog-kr/blog-us/blog-thematic 작성 시 reference로 읽음.

**갱신 정책**: `/blog-learn` 실행 시 모든 lesson을 다시 읽고 처음부터 재작성. 중복·모순 통합.

**우선순위**: 본 가이드 ↔ blog-{kr,us,thematic} SKILL.md 충돌 시 본 가이드가 우선.

**글 유형 분기 (가장 중요)**:
- **일일 리뷰(blog-kr, blog-us)** = 단정형 어미 "~임/~됨/~함", 압축 호흡
- **테마 deep-dive(blog-thematic)** = 존댓말 "~입니다/~있습니다", 서술 호흡

---

## 삭제 (이런 건 빠진다)

- **시장 뉴스 표는 어닝(개별기업) 위주 2~3행으로만** [일일 리뷰]. 국제·정치·지정학 행은 표에서 빼고 매크로·원자재 본문 라인에 녹임.
- **방향이 부정 일색인 섹터 그룹은 통째 삭제** [일일 리뷰]. Sell/Short 신호만으로 채워진 약세 섹터(예: 에너지 XOM·COP·BP·TTE 전원 Sell)는 → 인사이트까지 통째 컷. 종목 수와 무관하게 *방향이 부정 일색*이면 삭제.
- **종목 2~3개짜리 후순위 섹터 그룹 만들지 말 것** [일일 리뷰]. 메인 테마에서 멀거나 종목 수 부족하면 통째 생략.
- **그룹 안 약세 outlier 단일 종목 삭제** [일일 리뷰]. 긍정 신호 묶음에 끼인 약세 종목(예: SYM Short Sign)은 빼고, 그 종목을 가리키던 → 인사이트의 부정 절도 함께 삭제. 묶음은 Long·긍정 신호만 남김.
- **종합 판단 마지막 줄의 중복 트리거 제거** [일일 리뷰]. 매크로 종합·앞선 bullet에 이미 등장한 트리거(예: USD/JPY 엔캐리)는 마지막 모니터링 줄에서 중복 언급하지 말 것.
- **메인 섹터 그룹은 카테고리 엄격** [일일 리뷰]. 그룹 성격과 어정쩡한 종목(예: 반도체 AVGO를 하드웨어 그룹에)은 bullet에서 빼되, 종합 판단 산문에는 남겨도 됨.
- **데이터 한 줄 끝의 짧은 해석어 자제** [일일 리뷰]. "BITCOIN ... — 차익실현" 같은 단어 한두 개짜리 해석은 빼고 데이터만.
- **점수 표기 금지** [공통]. "매크로 28점" 같은 점수 언급 제거. 등급 단어만.
- **개별 매크로 항목 헤더의 [등급] 라벨 삭제** [일일 리뷰]. "1) 금리 ... [위험]"의 `[위험]`·`[주의]`는 뺀다. 등급어는 **매크로 종합 라인에만**.
- **종합 판단 마지막 액션의 구체 종목 괄호·과잉 부정 액션 삭제** [일일 리뷰]. 괄호 종목 나열과 "~추격 매수 자제"식 부정 액션을 덜어 간결화.
- **뭉뚱그린 "주도주 중심 대응 유지" 마무리 삭제 → 종목군별 구체 액션** [일일 리뷰]. 그날 핵심 종목군 각각의 행동 결론(예: AI부품 "축소 보류", 로봇 "ETF로 섹터 분산")으로 닫음. 일반론 한 줄로 뭉개지 말 것.
- **주도 섹터 → 인사이트의 평가절하 부연은 한 번만** [일일 리뷰]. 주도 섹터에선 "젠슨황 발언 트리거."로 짧게 끊고, 평가절하/테마주 논의는 종합 판단으로 내려서 한 번만 전개.
- **도입부 안내문 과한 대비 압축** [테마]. "무엇을 코어로 담고 무엇을 위성으로 가볍게 둘지" → "**무엇을 코어로 둘지**". 도입에서 코어/대비군 양쪽 다 예고하지 말고 코어 한쪽으로 압축(대비 구조는 본문 비중 섹션에서 전개).
- **`* 공통 패턴 N개` 별표 마커 삭제** [테마]. 마커 없이 자연 bullet 이어쓰기.
- **영문 제목·날짜 괄호 표기 삭제** [테마]. 한글 캐주얼 + 핵심 부제로.
- **표에서 "대체 베이스(분모)" 보조 행 삭제** [테마]. 카테고리당 핵심 1행만.
- **결론부 "예수금 5~10% 상시 확보" 명시 문장 삭제** [테마]. 원칙은 유지하되 매 글 문장으로 반복 X.
- **결론부 부정 액션 문장 삭제** [테마]. 회피 액션은 "함정" bullet에서 다뤘으면 비중 문단에서 중복 X. 늘리는/유지 액션 중심 마무리.

## 어조 (이렇게 다듬는다)

- **일일 리뷰 종합 판단 = 단정형 어미** — "~ 트리거.", "~ 배경이 됨.", "~ 작용하고 있음.", "~ 부분임.", "~ 필요가 있음."
- **한정 부사 "만"으로 대비 강조** [일일 리뷰]. "초장기물 소폭 상승" → "초장기물만 소폭 상승".
- **이중조건 템플릿 회피** [일일 리뷰]. "포트폴리오를 다양하게 구성한다면 ~지만, 주도주 중심이면 ~" (A) 모드 X. 다단락 흐름형으로.
- **초강세 후 1일 급락 = "부정 단정" 회피, "단기 sell sign"으로 완화** [일일 리뷰]. 직전까지 초강세였던 종목(예: 삼성전기·LG이노텍)의 단발 급락은 "주도 자리가 깨졌다=부정적" 단정 대신 **"최근 초강세를 보였던 ~은 단기 sell sign으로 볼만한 하락"**. 진위는 **익일 회복 여부로 확인, 그때까지 축소 보류**. (단, 추세적 약세 섹터의 Sell/Short은 그대로 노이즈 처리 — 위치별 차별화는 유지)
- **테마글 = 존댓말 + 서술형 풀어쓰기** — "조정 국면입니다", "고민을 해야할 시점에 놓였습니다". 일일 리뷰 단정형과 구분.
- **친근한 호기심 자극 제목** [테마]. 질문형 + 캐주얼 한국어 본제 + 분석 키워드 부제.
- **제목 본제 = 본문이 실제로 푸는 가장 구체적인 인과 퍼즐** [테마]. 일반적 시장 상태 묘사보다 구체 사건·고유명사 인과형 질문으로. **단 사용자가 이미 제목성 프레이밍을 줬으면 그걸 살림**(인과 퍼즐 규칙은 사용자가 제목 안 줬을 때만).
- **테마 제목 = 둥근 캐치 숫자 + 현상 개념어 부제** [테마]. 본문 정밀 수치와 달리 제목은 라운드 훅 숫자 허용. 부제는 한 단어 현상 라벨.
- **독자 가치 제안 한 문장** [테마, 도입부]. 독자의 현재 의사결정 상황 명시.
- **상승 catalyst 헤딩 = "쇼크" → "서프라이즈"** [테마]. 상승 방아쇠 소제목엔 긍정·중립어.
- **결론 종결 = 부드러운 안내형** [테마]. "~포트폴리오를 운영하면 되겠습니다"처럼 권유로.
- **도입부 강조 수식 보강** [테마]. "상승장" → "대세 상승장".
- **매크로 종합 라인에 등급 단어 유지** [일일 리뷰]. "매크로 종합 **주의** — 고금리..."처럼 ★ Macro 종합 등급(주의/위험)을 살림. 점수는 빼고 등급어만.
- **시장 폭 약화는 본인 매매 맥락으로 한정** [일일 리뷰]. "강한 것만 오르는 구간" → "~ **추세추종으로 매매 시** 포트폴리오 안정성은 줄어드는".
- **최상급/1등 표현 회피는 테마글에도 적용** [공통]. "로봇이 **가장** 강력한 후보" → "**강력한 후보 중 하나**". 단정적 1등 단언 대신 후보군 위치로.
- **테마글 매매 함의 = 진입 타이밍 지시보다 종목 성격 규정** [테마]. "추격보다 눌림 분할 접근이 유효" → "아직은 **주도주보다는 테마주의 성격으로 바라볼 필요**". 구체 진입 타이밍 처방 대신 성격 판단으로 톤다운.

## 표현 (이렇게 치환한다)

- **시장 뉴스 표 분류 = 단일기업이면 "개별기업", 섹터/테마 전반을 움직였으면 "섹터"·"반도체"** [일일 리뷰]. 뉴스가 한 종목이 아니라 섹터 전반 모멘텀이면 분류 칸에 섹터명을 쓰고, 이벤트 칸엔 종목군 호칭("로봇주", "2top"). 어닝 행도 "어닝"은 분류명으로 쓰지 말 것(시트 분류 체계와 일치).
- **수급은 추정 절대 금지 — 시트 정확값** [일일 리뷰, 반복 위반 항목]. 뉴스 헤드라인 어림수(예: "외인 8조 팔자")를 그대로 쓰지 말 것. 실제는 외국인 -6.6조 / 개인 +6.4조 / 기관 **+0.2조(매수)**였던 사례 — 부호까지 틀릴 수 있음. (앞선 사례: 코스닥 외국인 +8,016억 추정 → 실제 +7,943억). 종합 판단에 옮겨 쓴 숫자도 함께 정정.
- **순우리말 트리거어 → 외래어 "트리거"** [공통]. "방아쇠는 ~ 방한입니다" → "**트리거는** ~ 방한입니다". ("직격탄"→"직격탄으로 해석" 류 결.)
- **포트 대비군 = "보조(주)" 고정, "위성" 금지** [공통]. "순수 메이커는 위성으로 가볍게" → "**보조로** 가볍게". 코어(계층)+핵심 주도주(수식) ↔ 대비군은 "보조주".
- **분리 어휘 → 통합어** — "비철·은" → "산업 금속".
- **데이터 2~3개 묶음 → "내러티브"** — "디스인플레+장기 금리 미세 하락" → "디스인플레 내러티브".
- **메타 동사 "정렬" 자제** — "양대 어닝이 ~로 정렬" → "~의 상향 가이던스로".
- **압축 동사 → 서술 동사** [테마]. "손꼽혀온" → "상승 가도를 달려오던".
- **분류·평가어 한 단어 끼우기** [테마]. "7~17% 고점대비 조정" → "7~17% 고점대비 정도의 안정 조정".
- **증액/감액 트리거 = 추세추종 신호 용어** [공통]. "비중을 늘리는 트리거" 대신 "~으로 **long sign** 발생 시 추가 증액", "~뉴스 플로우에 유의(**Sell sign** 대비)".
- **포트 계층 = "코어"(계층) + "핵심 주도주"(수식) 병기** [테마]. 대비군은 "보조주".

## 구조 (이렇게 재배치한다)

- **5섹션 = 1.시장 뉴스 → 2.매크로 → 3.지수와 수급 → 4.섹터 → 5.종합 판단** [일일 리뷰]. 경제지표는 별도 번호 없이 1번 시장 뉴스 표에 행("경제지표")으로 통합, 표 아래 `→` 한 줄로 3축 해석. 발표 없으면 행 생략.
- **수급 = 코스피 + 코스닥 둘 다, 3주체 정확값** [일일 리뷰]. "코스피: 외국인 [-6.6조] / 개인 [+6.4조], 기관 [+0.2조]" + "코스닥: 개인 [-4,091억] / 외국인 [+3,060억], 기관 [+1,285억]" 두 줄. **수급 주체 배열 = (매도) 먼저 / 슬래시 / 같은 방향끼리 콤마**. 1조 미만은 억 단위.
- **일일 리뷰 제목 = 영문 + 괄호 날짜** — 미장 "Global Market Review (YYYY년 M월 D일)", **국장 "KOR Market Review (YYYY년 M월 D일)"**. (한글 제목 X. 테마글은 정반대로 한글 캐주얼)
- **일일 리뷰 제목의 괄호 날짜 = 실제 거래한 세션 날짜(현지)** [일일 리뷰]. KST 발행일·시트 날짜 아님. 미장은 시트가 260603(6/3 KST 생성)이어도 데이터가 6/2 마감분이면 제목은 **"6월 2일"**. (draft를 KST 날짜로 달면 발행 시 세션 날짜로 정정됨)
- **테마글 역사 사례는 연대순 정렬** [테마]. 과거 사례 나열은 최신순·임의 순서가 아니라 **오래된 것 → 최근** 시간순(예: 2020 테슬라 → 2023 엔비디아 → 2025 깐부치킨).
- **종합 판단 = 다단락 흐름 (이중조건 X)** [일일 리뷰]. 인과 사슬: ①매크로 우호 요인 통합 ②잔존 부담 한 줄 ③어닝/이벤트 트리거 + 메인 테마 ④한 가지 고민(양면성) ⑤본인 액션 + 모니터링(앞서 쓴 트리거 중복 금지). KR은 bullet 골격을 쓰되, 그날 핵심 종목군(초강세 급락주·뉴스 테마주 등)은 **각각 독립 단락으로 펼치고 단락마다 구체 액션으로 닫음**.
- **종합 판단 마지막 = 외부변수 양시나리오 병렬 마무리** [일일 리뷰]. 진행 중 외부 변수가 있으면 진전/결렬 양갈래 인과사슬로 닫음. **단 발표·종료된 과거 이벤트를 워치로 재탕 금지** — 워치는 "당일 기준 실제 임박(다음 1~2일)" 이벤트만, 없으면 워치 자체 생략.
- **핵심 인사이트 구절 볼드 처리** [일일 리뷰]. 각 → 인사이트와 종합 판단의 핵심 명사구를 `**볼드**`.
- **테마글 도입부 = 4~5문장 펼침** — 양상 묘사 + 폭 비교 + 독자 가치 제안 + 글 안내.
- **테마글 데이터 표 끝 = 데이터 요약 + 캐주얼 1차 가설** — "단순히 생각하자면 ~ 일 수도 있음".
- **도입부 지수 레벨 = 발행 시점 실시간 값으로 갱신** [테마]. 종가 % 박제 X, 서술형 최신 레벨.
- **테마글 역사적 사례 = 별표 마커 없이 공통 패턴 이어쓰기**.

## 사실추가 (이런 맥락이 보태진다)

- **인트라데이 사실 본문 보강** [일일 리뷰]. 사용자가 아는 장중 흐름을 매크로·지수 코멘트에 직접 녹임 — "금리 장중 1.5%대 상승까지 갔으나 마감 소폭 반락", "달러 최고가에 근접중", "코스피 장중 7,000 터치 후 7,200 회복" 등.
- **시장 뉴스 표 아래 요약 해석 한 줄** [일일 리뷰]. 표를 닫는 종합 해석 — "이상의 반도체 섹터에 긍정적인 뉴스 플로우가 지속되며 상승을 뒷받침."
- **종목군별 구체 액션** [일일 리뷰]. 초강세 급락주 → "단기 sell sign, 익일 회복으로 진위 확인, 축소 보류". 뉴스성 테마주 → "재료 소멸 후 강세 유지 모니터링 + 개별종목 추격 대신 ETF로 섹터 전반 분산"(예: ISA 계좌 로봇 ETF).
- **인지도 낮은 종목 bullet에 `- 사업영역` 꼬리표** [일일 리뷰]. "DDOG ... - 서비스 모니터", "CRWD ... - 사이버 보안". 인지도 낮은 종목만 선별.
- **매크로 4~5지표 묶어 "재랠리/조정 배경" 평가** [일일 리뷰]. 산발 나열 X, 종합 판단에서 한 단락 묶음.
- **시장 폭·포트폴리오 안정성 trade-off 메타 인식** [일일 리뷰]. 시장 폭 좁아짐과 안정성 약화를 명시.
- **추세 맥락 + 모니터링 워치 페어** [일일 리뷰]. 추세 한 마디 + 워치 포인트를 본인 액션과 짝지음.
- **주도 섹터 확산 강세의 트리거가 뉴스성이면 평가절하 단서** [일일 리뷰]. AI 밸류체인 동반 강세를 무조건 긍정으로 보지 말고, 촉매가 단발 뉴스(예: 젠슨황 방한)면 "**펀더멘탈보다 뉴스에 의한 것이라 가치는 약간 평가절하**, 일종의 테마주" 단서.
- **독자 행동 유발 한 문장** [테마, 도입부]. 독자가 이 글로 답하게 될 의사결정 질문 명시.
- **테마 제목 = 구체 사건 명사화** [테마]. "젠슨 황이 한국에 **온다는데**"(막연한 예고형) → "젠슨 황의 **한국 로봇 샤라웃**"(구체 사건 캐치 명사). 부제(밸류체인 지도·옥석)는 유지.
- **테마글 결론 비중부 = "ETF 광범위 진입 → 주도 확정 후 핵심주 격상" 2단계** [테마]. 막연한 "코어 중심 운영" 마무리 대신 **초기엔 섹터 ETF로 넓게 진입 → 주도 섹터 자리매김 후 가장 센 종목을 핵심 주도주로 편입해 무겁게**의 2단계 액션으로 구체화.
- **수급 사실에 시장 대비 맥락 + 부드러운 평가어** [공통]. "단순 개인 테마가 아니라는 신호" → "**시장 전반적인 매도 속에서도 이 섹터는 매수한다는 점은 고무적**". 외국인 순매수 등 수급 팩트는 시장 전체 흐름 대비로 의미를 짚고 "고무적" 류 톤으로 닫음.
- **증액 시점에 "주도주 격상 고민" 명시** [공통]. "long sign 유지·재점화 종목에 증액" → "증액**(이 때 비로소 주도주로의 격상을 고민)**". 증액 = 보조→주도 격상 판단 시점임을 괄호로 명확화.
