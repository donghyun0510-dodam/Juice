## 분류된 패턴

### 삭제
- **매크로 → 의 추정 해석 통삭.** 금리 → "인상에도 3년물은 소폭 하락. **인상이 선반영된 것으로 보임**." → 뒷문장 삭제, 사실만 남김. 단 같은 판단이 종합 판단에는 "인상 자체는 선반영됐으나"로 **살아있음** — 즉 삭제가 아니라 **해석의 배치 이동**(매크로 →는 사실, 해석은 종합). (③ 진정성 / §2.3 · §4 중복 배치)
- **하락 종목의 Long Sign은 강세 근거 목록에서 컷.** "long sign은 S-Oil·하나금융·**티에스이** 셋뿐" → "S-Oil·하나금융뿐". 티에스이는 -0.70% 하락 종목이라 Long Sign이 점등해도 그날 강세의 근거로 세지 않음. **섹터 → 와 종합 판단 양쪽에서 일관 삭제.** (③ / §6 신호 위계)
- **평가 꼬리 절단.** "한화오션의 상승도 Short Cover Sign 수준**이라 의미를 부여하기 어려움**." → "Short Cover Sign 수준." 신호 등급만 밝히면 평가는 자명하므로 꼬리를 붙이지 않음.
- **원자재 → 의 평가어 삭제.** "$80 아래에서 유지되는 부분은 **그나마 긍정적**" → "유가 급등에서는 조금 진정되는 추세". 평가를 빼고 추세 묘사로 바꾼 대신, **매크로 종합 라인에 "유가의 상승이 부담"을 추가**해 평가를 종합이 전담하도록 옮김. (§4 중복 배치 — 등급 평가는 매크로 종합 라인)

### 어조
- **확인 안 된 개별 인과는 헤지 어미로.** "국산 모듈에는 우호적." → "국산 모듈에는 우호적**이라는 판단도 있음**." 한화솔루션 낙폭↔태양광 정책 연결이 섹터 전반 하락과 뒤섞여 단정 불가한 사안이라, 본인 단정이 아니라 전언체로 물러섬. (③ 진정성)
- **추상 판정 → 가격 위치 사실.** "반등을 이끌던 반도체가 다시 무너지며 **재랠리 전환은 무산**." → "…무너지며 **하단까지 내려옴**." 신호·국면 라벨("재랠리 전환 무산")보다 가격이 어디까지 갔는지의 사실을 선호. (§0.6 메타 라벨 회피의 연장)

### 표현
- **표 이벤트 칸 추가 압축**: "한은 금통위" → "**금통위**"(주체 접두 제거), "반도체 고점론" → "**반도체**"(-론 같은 성격 접미도 제거). §2.2의 "접두 수식·부연 없는 짧은 토픽" 규칙이 예상보다 더 짧은 쪽으로 적용됨.

### 구조
