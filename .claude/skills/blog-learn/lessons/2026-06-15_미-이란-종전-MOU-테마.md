## 분류된 패턴

(이 글은 작성 세션 중 사용자와 여러 차례 수정을 거쳐 draft를 발행본에 이미 근접시킨 케이스라, 발행 단계 순수 변경은 4곳으로 소수. 그만큼 남은 4곳은 "사용자가 끝까지 손보는 지점"이라 신호가 강함.)

### 삭제
- **본문 인라인 영문 직역 병기(괄호) 삭제** [테마]. "유가 스파이크를 압도**(tech rally overpowers oil spike)**" → "유가 충격을 압도". 한글 본문에 끼운 영문 원문 병기는 발행 시 제거 — 한글 표현만 남김. (기존 "영문 제목·날짜 괄호 삭제"를 **본문 인라인 영문**까지 확장. 영어 인용구로 권위를 더하려는 충동을 누르고 한글로 단정.)

### 어조
- **테마 제목 동사 = 구체 사실어 → 생동감 모션 동사** [테마]. "전쟁 중에도 **신고가였던** 증시" → "전쟁 중에도 **달리던** 증시". 제목의 구체 사실("신고가")은 본문에서 충분히 풀리므로 제목은 캐주얼·생동감 동사로 바꾸고, 본문에서 반복되는 핵심어("신고가")를 제목에서 회피해 중복을 던다.

### 표현
- **외래어 → 한자어 치환** [공통]. "유가 **스파이크**" → "유가 **충격**". (cf. "스파이크/롱 사인" 류 외래어는 본문에선 한자어·한글로, 음성 대본에선 음가로.)
- **시간축 단계 요약의 추상명사 → 구체 결과 명시** [테마]. "전쟁 발발 **충격**(2~3월)" → "전쟁 발발 **쇼크로 인한 급락**(2~3월)". 단계별 흐름 bullet에선 추상 명사(충격)보다 실제 시장 결과(급락)를 붙여 구체화. (일일 리뷰의 "강조어 '쇼크' 삭제"와 방향이 반대 — 거긴 괄호 트리거가 이미 있어 '쇼크'가 중복이라 삭제, 여긴 시간축 bullet이라 '급락'이 정보 추가. 맥락으로 구분.)

### 구조
- 변화 없음(섹션·헤딩·표·bullet 골격 그대로 발행).

### 사실추가
- 신규 데이터·인용 추가 없음.

### 결론·액션 (이번 lesson 핵심)
- **테마 결론 하방 대응 = "방어 섹터(에너지·금) 로테이션" → "현금 비중 확대"** [테마/공통]. 전략 마무리에서 Sell sign·하방 트리거 발생 시 대응을 "유가 하락 수혜주부터 줄이고 **에너지·금 쪽으로 무게를 옮기는 방어 전환**"이 아니라 "유가 하락 수혜주부터 줄이고 **다시 현금 비중을 늘려야**"로. 하방 국면에서 다른 섹터로 갈아타기보다 **현금화가 본인 규율**(롱 온리 + 리스크오프 시 현금 누적 확대 원칙과 정합). 테마글이라도 하방 액션은 "방어 섹터 로테이션" 권유로 닫지 말 것 — 현금 비중 확대로 닫는다.
