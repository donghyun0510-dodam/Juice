> 맥락: 작성 중 인과를 3연속 짜맞춤(①미실현 과세 토론회 트리거 ②미 반도체 폭락 전이[검색 환각] ③숏감마 -10% 증폭 서사). 사용자가 "결과에 끼워맞추지 말고 사실만"이라 지적 → 최종 draft를 "확인된 사실 / 언론이 든 요인 / 단정 불가"의 임상적 구조로 과잉 후퇴. **발행본은 둘 다 아닌 중간** — 수급 하드데이터로 뒷받침되는 인과는 헤지하여 단정, 추측성 뉴스는 "언급 후 강등", 검증 안 된 메커니즘은 삭제.

## 분류된 패턴

### 삭제
- **검증 모호한 기술적 메커니즘은 통째 삭제** — '숏감마(단일종목 레버리지 ETF)'·'서킷브레이커 시각(2:33)'·'역대 최대 낙폭(-910p)'을 본문 전 군데에서 전부 컷. 그럴듯해도 그날 -10%의 원인으로 *내가 합성한* 메커니즘은 남기지 않음.
- **검색 환각發 미검증 인과 삭제** — "미국 반도체주 폭락 전이"(검색 AI 요약 환각, 실제 미 반도체는 폭락 안 함)는 본문에서 완전 제거. 원문 fetch 교차검증으로 걸러야 했음.
- **시장 뉴스 표에서 추측성 요인 행 삭제 → 종합 판단으로 강등** — 'MSCI 불발·과세 논의', '반도체 경계심리(마이크론)' 행을 표에서 빼고, 표는 연준 긴축(강달러→외인매도)·부동산 세제·삼성전자·경제지표 4행으로 단순화. 추측성 센티 요인은 표가 아니라 종합 판단 한 문장으로.
- **"확인된 사실 / 언론이 든 요인 / 단정 불가" 임상적 메타구조 삭제** — 내가 과잉 유보로 만든 3분할 라벨 나열을 발행본은 흐름형 해석 bullet로 되돌림. 사실을 라벨로 분류해 흩뿌리지 말고 단정형 해석으로 풀 것.
- **과잉 유보 문구 삭제** — "한국만 -10% 빠진 이유는 명확히 단정하기 어려움" 같은 *전면 유보*는 삭제. 수급 데이터가 방향을 가리키면 유보로 도망가지 않음.

### 어조
- **하드데이터(수급) 근거 인과는 헤지하여 단정 — 전면 유보 금지** [핵심]. 외국인·기관 8.7조 동반 순매도는 검증된 사실 → "수급발 패닉 / 외국인 디레버리징 집중된 것으로 보임 / 포모로 해석됨"처럼 **헤지 어미(~로 보임/해석됨)로 단정**. "원인 단정 불가"로 후퇴하는 것도 과오. (역산 금지 ↔ 과잉 유보 금지의 균형 = [[feedback_no_causal_fitting]] 보강)
- **추측성 뉴스는 "언급 후 강등"** [핵심]. 미실현 과세·MSCI 불발 같은 센티 요인은 빼지도 띄우지도 말고, "센티 흔들만한 뉴스 많았으나, **그보다는** [수급·환경]"으로 명시적 종속. 누락(무시)도 과대평가(주범화)도 아닌 제3의 처리.
- 매크로 인사이트 강도 강화: 환율 "외국인 매도를 **부추긴** 배경" → "외국인 자금 이탈의 **직접** 배경".

### 표현
- 금리: "안전자산 선호로 **채권 매수**, 단기 금리 반락" → "안전자산 선호로 **채권 가격 상승**"(부연 압축, 표준어).
- 원자재: "**위험회피 국면에** 유가도 동반 하락" → "**중동 긴장 완화로** 유가 하락했으나 **시장에 영향은 미미**"(그날 실제 유가 트리거+영향도 평가로 교체).

### 구조
- **종합 판단 = 임상적 3분할(사실/요인/대응) 아닌 흐름형 해석 bullet** — 발행본은 ①수급발 패닉+디레버리징 read(+센티뉴스 강등 문장) ②매크로 비(非)트리플+질 악화 ③대응. 사실 나열 라벨 대신 "사실. → 해석" 흐름.
- **표 → 라인 = 그날 가장 직관적인 단일 포인트 + 임박 이벤트** — "서킷브레이커/언론 요인 정리(단정 불가)"(메타) 대신 "삼성전자 HBM4 호재에도 -12% = 수급발 매도 시사 + 마이크론(6/24) 경계 심리"(구체).

### 사실추가
- **마이크론 6/24(목) 실적 경계심리** 추가(표 → + 종합 대응 워치) — 미 반도체 폭락이 아니라 *마이크론 개별 실적 앞둔 경계*가 실제 사실.
- **니케이 큰 하락(-3.55%)을 디레버리징 근거로 사용** — "니케이의 하락도 큰 것으로 보아 … 외국인 디레버리징 집중"으로 주변국 데이터를 인과 근거로.
- **breadth 갈림 ↔ 수급 연결 명시** — 수급 →에 "아시아 타국에 비해 한국의 하락이 심했던 배경에는 외국인 매도 ↔ 원/달러 강세가 직접 연결" 추가.
- 핵심 명사구 볼드 추가("사실상 상승 섹터 부재", "주도주…두 자릿수 훼손").
