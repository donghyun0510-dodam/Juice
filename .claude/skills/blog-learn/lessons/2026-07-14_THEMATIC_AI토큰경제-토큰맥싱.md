## 개요

첫 **개념·용어 정리형 테마글**(특정 IPO/정책/사건이 아니라 "토큰이란 무엇인가"를 해설). draft가 **거의 그대로 발행**됨 — 의미 있는 텍스트 변경 3곳뿐. 개념 해설형 테마글의 골격(개념→경제 단위→신조어→최적화→역사 프레임→투자 함의 6섹션)과, **종목별 하드 비중 없이 시장 레벨 투자 함의로 닫는 마무리**가 그대로 수용됨.

## 분류된 패턴

### 삭제
- 없음. draft에서 잘려나간 문장·표현 없음.

### 어조
- §3 결론 어미: "허영 지표에 **가까웠던 셈입니다**"(본인 단정) → "허영 지표에 **가깝다는 평가를 받게 된 이유입니다**"(외부 평가·전언체). 본인이 직접 규정하는 대신 업계 평가로 돌림 (③ 진정성 — 미검증 가치판단은 전언 처리).

### 표현
- §3 신조어에 따옴표 부여: "흐름은 **토크노믹스**로" → "흐름은 "**토크노믹스**"로". 처음 등장하는 신조어를 따옴표로 명시 (④ 가독성).
- §5 `\* 공통 패턴` → `* 공통 패턴`. draft에서 별표를 이스케이프(`\*`)한 것을 발행 시 제거. **마크다운 이스케이프 금지 규칙 재확인**([[feedback_blog_draft_save_rule]]) — draft 단계에서 `\*`·`\)` 넣지 말 것.

### 구조
- §2 표 아래 → 인사이트에 주어 명시: "약 50배(98%) 하락했고" → "**백만 토큰당 가격은** 약 50배(98%) 하락했고". 수치의 대상을 명확히 (④ 가독성 / 숫자 명확화).
- 6섹션 순서·헤딩·표 전부 유지. 표 3개(가격 하락/최적화 기법/제번스 사례)·`→` 인사이트 구조 그대로 발행.

## 학습 요지 (5축 렌즈)
- **개념 해설형 테마글 포맷 검증(② 주제 일관성 / ① 전문성)**: IPO·정책 사건이 아닌 순수 개념/용어 정리글도, 도입부를 시황이 아닌 **핵심 인용(젠슨 황 GTC 발언)**으로 열고 → 개념 → 경제적 의미 → 신조어 → 최적화 → 역사 프레임 → 투자 함의로 전개하면 near-verbatim 통과.
- **투자 함의 처리(① 전문성 / ③ 진정성)**: 개념 해설형은 §6에서 **종목별 하드 비중(코어/위성 %) 없이** 시장 레벨 논쟁 프레임(수요 지속론 vs 효율·규율론) + **분수령 이벤트 시점**(하이퍼스케일러 실적 캘린더)으로 닫는 방식이 수용됨. 추세추종 관점("확인 후 조절")으로 마무리.
- **숫자 베이스 통일 검증(③ 진정성)**: 상충하는 수치(가격 50배 vs 1,000배)를 한 줄에 두 베이스 모두 명시("GPT-4급 기준 50배 / 특정 성능 도달 비용 기준 1,000배")한 처리가 그대로 통과.
