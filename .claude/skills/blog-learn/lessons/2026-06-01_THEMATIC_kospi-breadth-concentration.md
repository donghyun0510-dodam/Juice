## 분류된 패턴

### 삭제
- **시총·breadth 표에서 "대체 베이스(분모)" 보조 행 삭제** — "코스피 50.4%" 옆에 "전체 증시 46%(코스닥·코넥스 포함)" 같은 동일 메트릭 다른 베이스 보조 행은 컷. 카테고리당 핵심 1행만 남김. (기존 "숫자 일관성: 베이스 통일" 규칙의 실천형 — 두 베이스 병기보다 한 베이스로 정리)
- **결론부 "예수금 5~10% 상시 확보" 명시 문장 삭제** — 상시 적용되는 현금 운용 원칙은 매 글 문장으로 반복 박지 않음(군더더기). 원칙 자체는 유지하되 결론마다 명시 X.
- **결론부 부정 액션 문장("코스닥 비중 줄여 둠") 삭제** — 늘리는/유지하는 액션 중심으로 마무리. 회피 대상은 이미 "함정" bullet에서 다뤘으면 비중 문단에서 중복 서술 안 함.

### 어조
- **상승 catalyst 헤딩: "쇼크" → "서프라이즈"** — 상승 방아쇠를 다루는 소제목에 부정 함의 단어(쇼크) 대신 긍정·중립어(서프라이즈). (예: "마이크론 쇼크" → "마이크론 서프라이즈")
- **결론 종결 = 부드러운 안내형** — "~포트폴리오를 운영하면 되겠습니다"처럼 독자 가이드형으로 닫음. 단정·지시형보다 권유형.
- **도입부 강조 수식 보강** — "상승장" → "대세 상승장"처럼 한 단어 강조어를 얹어 현상의 위상을 또렷하게.

### 표현
- **증액/감액 트리거를 추세추종 신호 용어로 표기** — "비중을 늘리는 트리거는~/줄이는 트리거는~" 추상 서술 대신 "~으로 **long sign** 발생 시 추가 증액", "~뉴스 플로우에 유의(**Sell sign** 대비)". 사용자 매매 시스템(롱돌이 신호)과 어휘를 연결.
- **포트 계층 용어 = "코어"(계층) + "핵심 주도주"(수식) 병기** — 둘 중 하나로 치환하지 말 것. "코어는 ~를 핵심 주도주로 무겁게 싣되 … 코어 내에서 분산"처럼 코어를 계층 명사로 유지하고 핵심 주도주는 수식으로 함께 씀. 대비군은 "보조주".

### 구조
- **도입부 지수 레벨 = 발행 시점 실시간 값으로 갱신** — draft의 종가·일일 등락률 괄호 "(+3.55%)"는 떼고, "오늘도 급등하여 8,800선을 넘어서며"처럼 발행일 기준 최신 레벨 + 서술형. 정확한 종가 %를 박제하지 않음(발행 시점엔 이미 바뀜).
- **테마 제목: 사용자 원래 질문 프레이밍 우선** — 사용자가 요청에 이미 제목성 프레이밍("코스피 급등, 하지만 상승 종목은 200개 뿐?")을 줬으면 그걸 살림. draft에서 "구체 인과 퍼즐"형으로 갈아끼운 제목은 사용자 프레이밍으로 회귀됨. 구체 인과 퍼즐 규칙은 *사용자가 제목을 안 줬을 때* 적용.
- **제목 = 둥근 캐치 숫자 + 현상 개념어 부제** — 본문 정밀 수치(313개)와 달리 제목은 라운드 훅 숫자("200개")를 써도 됨. 부제는 한 단어 현상 라벨("대형주 위주 슬림화")로 닫음.
