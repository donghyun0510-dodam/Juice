### 삭제
- **시장 뉴스 표에서 다른 섹션(섹터)에서 전개되는 행 삭제** — draft "개별기업 / 유통 / 신세계·롯데쇼핑·현대백화점 동반 강세" 행이 발행본에서 통째 삭제(유통은 섹터 4번 주도 섹터에서 전개). 기존 규칙 재확인.
- **호르무즈 행 "유가·해운 자극" → "유가 자극"** — 해운은 같은 표에 별도 행(섹터/해운)이 있어 중복 제거. 한 트리거의 파급 대상이 표에 별도 행으로 있으면 상위 행에선 뺀다.
- **환율 인사이트의 레벨·경보단계 부연 삭제** — draft "원화 약세 지속, **1,530원 위. 환율 종합 경보 주의 단계.**" → 발행 "원화 약세 지속." 매크로 항목 코멘트는 데이터 + 핵심 한 마디로 끊고 레벨 부연·경보 단계 표기는 덜어냄.
- **매크로 종합 라인 괄호 부연 삭제 + "트리플"→"지속"** — draft "고금리(3년물 3.9%), 고환율(1,531원), 고유가(호르무즈 리스크) 트리플." → 발행 "고금리, 고환율, 고유가 **지속**". 종합 라인은 등급어 + 트리플 키워드만, 괄호 수치·"트리플" 명사는 뺌.
- **"평균 회귀" 메타 표현 반복 삭제 (지수·종합)** — draft 지수 인사이트 "중소형주의 리벤지, **평균 회귀의 흐름**" / 종합 "리벤지, **평균 회귀의 흐름**" 양쪽에서 "평균 회귀" 삭제, **"리벤지"만 유지**. 단 섹터(유통) 인사이트의 "평균 회귀성 강세" 수식은 발행본에 살아남음 → 평균회귀는 *메타 진단(지수·종합)에선 빼고* 섹터 수식 정도만, 한 글에서 반복 금지. (자동 남발 금지 기존 규칙 강화)
- **"네 마녀의 날 변동성" → "높은 변동성"** — 지수 인사이트에서 이벤트 고유명 압축(시장 뉴스 표·표 위 요약에 이미 등장).
