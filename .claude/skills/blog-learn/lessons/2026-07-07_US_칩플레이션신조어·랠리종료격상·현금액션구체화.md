## 분류된 패턴

### 삭제
- **매크로 종합 라인의 인과 수식 절 삭제 → 데이터 사실만** (③ 진정성, ④ 가독성). "금리 전 구간 상승과 유가 강세**가 겹쳐 AI 성장주엔 역풍**, 위험자금이…" → "금리 전 구간 상승과 유가 강세. 위험자금이…". 매크로 종합은 데이터 나열 + 흐름 한 마디로 닫고, "~에 역풍" 같은 성장주 영향 해석은 종합 판단이 전담(매크로 종합엔 중복 X).
- **섹터 → 인사이트의 추측성 자금이동 해석 삭제** (③ 진정성). 에너지 → "정유·메이저가 동반 상승. **AI에서 빠진 자금의 이동처.**" → "…동반 상승."에서 끊음. 유가 강세→정유 상승은 사실이나 "AI 자금이 여기로 왔다"는 자금 흐름 단정은 추측이라 컷. (자금 로테이션 서사는 종합 판단에서 시장 구도로만.)

### 어조
- **주도주 Sell 다발일 = 완충/안도 프레임("순환매로 소화") 폐기 → 랠리 종료 우려로 격상** (③ 진정성). draft "지수 낙폭은 제한됨(다우 -0.25%). **순환매로 소화되는 국면.**" → 발행 "…제한됨. **4월 이후로 이어온 랠리의 종료 가능성이 높아지는 하락으로 보임.**" breadth가 양호(에너지·S/W 방어)해도, 본인이 실제 방어 액션을 취한 날은 종합을 "소화 중"으로 안심시키지 않고 랠리 종료 리스크로 단정. (line 74 "주도주 Sell 다발 → 어조 격상"의 US 재확인 — 단 여기선 완충을 함께 두기보다 랠리 종료 쪽으로 무게를 더 실음.)
- **"국한되고" → "집중되고"**. 하락 범위를 좁게 안심시키는 뉘앙스("AI에만 국한") 제거, 사실 묘사("AI에 집중")로.
- **"되돌림" → "하락"**. 섹터 → 마무리에서 "AI 트레이드 전반의 되돌림"(일시 조정 뉘앙스) → "AI 트레이드 전반의 하락"(담백·중립). 큰 하락일엔 "되돌림"이 완화어로 읽혀 격하.
- **final → 조건부 간결 액션 → 상태 지속 기반 방어 운용**. "추가 확산 시 축소" → "현재 위험 상태가 지속된다면 방어적인 포트 운용이 필요". 하루짜리 조건이 아니라 위험 상태 지속을 방어 근거로.

### 표현
- **시장 뉴스 국제 이벤트명 = 개념 압축 신조어로** (①·④). "AI Capex 지속가능성" → **"칩플레이션"**(칩+인플레이션). 메모리값 급등이 원가·수익성을 압박하는 현상을 한 단어 신조어로 명명 → 스캔성·전문성 동시 강화. (내용 셀은 그대로 두고 이벤트명만 신조어로.)
- **"삼성전자 2분기 실적" → "삼성전자 2분기 잠정 실적"** (③ 정확성). 잠정치(가이던스)임을 명시 — 확정 실적과 구분.
- **워치 대상 갱신 "TSMC·인텔·삼성전자" → "TSMC·인텔 등 반도체와 빅테크"**. 이미 발표된 삼성전자는 빼고, 앞으로 나올 빅테크 실적을 워치에 추가(미래 확인 이벤트만 남김).

### 구조
- **실매매 액션일 종합 판단 = (D+E) 다층 fact→action** (①·③). draft는 `-` 3개 + 마지막 `→` 1개였으나, 발행은 **bullet 사이에 `→` 현금 액션을 삽입**해 fact→action 페어가 두 번(중간: 현금% 축소 액션 / 마지막: 방어 운용 워치). 실제 포지션을 움직인 날은 액션 `→`를 종합 중간에 박아 사실과 짝지음.

### 사실추가
- **실제 포트폴리오 액션을 구체 현금%로 명시** (①·③·⑤). draft는 방향("방어적 유지·확산 시 축소")만 뒀으나 발행은 **"전일 국장 비중 축소(현금 55%까지)에 추가, 글로벌 증시도 Sell Sign 발생 종목 축소(max 현금 65%)"**. 본인이 실제 매도한 날은 % 하드값을 종합에 명시(내가 draft에서 "% 미명시"로 뺀 것을 사용자는 실제 액션 %로 채움). 교훈: **실매매를 취한 날은 현금 비중 %를 구체적으로 쓴다** — (E) 액션 마무리의 핵심. [[project_cash_path]]
