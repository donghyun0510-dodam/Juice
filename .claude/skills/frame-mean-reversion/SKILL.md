# 평균 회귀 프레임

## 핵심 프레임 — "평균 회귀 장세"
주도주 위주 장세에서 벗어나 그동안 소외됐던 섹터·종목이 동반 상승하는 흐름을 **"평균 회귀 장세"**로 명명.

단순한 큰 폭 상승이 아니라 "주도주 훼손 여부"를 같이 평가:
- 주도주(반도체·IT H/W·인터넷·2차전지 등) **상승 동반** → **부정적 시그널 아님**
- 주도주가 상대적으로 약하고 소외주만 튀어오름 → 모멘텀 피크아웃 경계

종합 판단 첫 bullet에 자주 배치, `→` 하위 인사이트로 "주도주 상승의 훼손이 없는 평균 회귀 장세는 부정적 시그널은 아님" 형태 강조.

## 부정형 사용
큰 폭의 *하락* 장에서 "평균 회귀"를 **역으로 사용**해 거짓 위안 차단:
"주도주 동반 훼손으로 평소의 '평균 회귀 장세'로 **폄하할 하락은 아님**".

즉 평소 큰 폭의 상승을 "평균 회귀"로 가볍게 보듯, 큰 폭의 하락도 "평균 회귀의 반대 작용"으로 가볍게 볼 유혹이 있는데, 주도주가 같이 무너지면 그 프레임을 적용하지 말라는 신호. Sell sign 다발 + 매크로 트리플 쇼크가 같이 오면 진짜 패닉으로 해석.

## 부정형 자동 적용 X
큰 폭의 하락에서 "평균 회귀 부정형"이 항상 적용되는 것은 아님. 사용자는 그날 분위기를 **단순히 "주도주와 함께 하던 섹터들의 동시 약세 = 시장의 질 약화"**로 짧게 끊는 경우도 많음. 매크로 트리플 위험·Sell 다발이 명확하면 굳이 평균회귀 프레임을 거치지 않고 "질 약화"로 직행. 평균 회귀 부정형은 사용자가 직접 그 단어를 꺼낸 날에만 활용.

## 긍정형도 자동 적용 X
큰 폭의 *상승*장의 긍정형 "평균 회귀 장세" 프레임도 **자동 적용 금지**. 주도주 동반 + 소외주 동반 두 자릿수 상승의 전형적인 큰 폭의 강세장이어도, 사용자는 "평균 회귀 장세"라는 단어를 직접 꺼내지 않는 경우가 많음. 대신 **사실 묘사 + 가격 레벨**로 짧게 끊음 — 예:
- "코스피 7천선까지 후퇴하던 약세를 극복하는 상승"
- "주도주, 시총 상위주 위주의 상승으로 그 외의 섹터는 소외됨"

메타 프레임("평균 회귀"·"새로 깨진 자리")보다 **직설적 사실 + 폭의 비대칭 묘사**가 사용자 시그니처. 사용자가 직접 단어를 꺼낸 날에만 활용.

## 역방향 적용 — 주도주 하락 + 약세 섹터 강세 = "의미 축소"
큰 폭의 하락이 아닌 **분화 장세**에서, 주도주(반도체·고듀레이션 자산)가 하락하는 가운데 **그동안 상대적/절대적으로 약세였던 섹터**(소프트웨어 일부, 디펜시브 등)가 강세를 보일 때:

"그동안 상대적/절대적 약세였던 섹터의 강세는 긍정적으로 보이지 않음", "주도주의 하락으로 **의미가 축소됨**".

즉 약세 섹터의 단독 강세는 평균 회귀 식의 모멘텀 회복 신호가 **아니라** 주도주 자금이 빠져나간 도피처일 수 있어, 시장 전체 긍정 시그널로 해석하지 않음. 종합 판단 bullet에서 "상승했으나 ~로 의미가 축소됨" 형태로 명시.

## 적용 가드
- 사용자가 그 단어를 직접 꺼낸 날에만 적용
- "광폭" 어휘는 사용자가 평소 사용 X — "상승"·"급등"·"큰 폭의 상승" 등 직설 표현 사용
