# 도담아빠 — 커뮤니티용 평문 변환 규약

표를 못 쓰는 커뮤니티(네이버 카페 등)에 붙여넣을 수 있도록, 당일 일일 리뷰 글의 **마크다운 표·헤딩·볼드를 평문으로 변환**한다. 일일 리뷰 작성은 `/blog-kr`·`/blog-us`, 학습은 `/blog-learn`.

## 핵심 원칙
- **원본을 재입력하지 말 것.** 긴 종목 리스트를 다시 타이핑하면 오타가 난다 — 반드시 변환 스크립트로 원본 파일을 그대로 읽어 변환한다.
- **내용은 그대로, 형식만 평문화.** 문장·수치·해석은 손대지 않고 마크다운 마커만 푼다.

## 동작 순서

### 1. 변환 스크립트 실행 (기본)
```bash
cd "C:\Users\dongh\Desktop\주식\AI agent"
python ".claude\skills\blog-community\convert_community.py" [YYYY-MM-DD] [KR|US]
```
- 인자 없으면 `published/` 폴더의 **가장 최근 `_published.md`** 자동 선택(없으면 `draft/`).
- 날짜·시장 인자는 순서 무관(`/blog-community US`, `/blog-community 2026-06-17 US` 모두 가능).
- 특정 파일 직접 지정: `python convert_community.py --file "<경로>"`
- **라벨 (기본 ON)**: 시장 뉴스 각 행 앞에 `[이벤트]` 라벨 표기(예: `[레버리지 ETN] 금감원…`, `[삼성전자] 하나증권…`). 평문에서 이벤트명이 내용에 안 드러나도 명확. **`--no-labels`**로 끄면 내용만 출력.
- 출력: stdout에 변환 결과 + `blog\community\YYYY-MM-DD_{KR|US}_community.md` 저장.

### 2. 변환 규칙 (스크립트가 자동 적용)
- **헤딩 마커 제거**: `## 1. 시장 뉴스` → `1. 시장 뉴스`, `### 1) 금리` → `1) 금리`, `# 제목` → `제목`
- **볼드 제거**: `**텍스트**` → `텍스트`
- **표 → 평문**:
  - **지수/구분/지표 전치표**(헤더행 + 값행) → 인라인 슬래시: `DOW -0.98% / S&P500 -1.21% / NASDAQ -1.34% / RUSSELL2000 -0.72%`
  - **시장 뉴스표**(분류·이벤트·내용) → 각 행을 **문단**으로. 분류가 `경제지표`면 `경제지표 발표: {내용}`, 그 외 행은 기본적으로 `[{이벤트}] {내용}`(라벨 ON 기본). `--no-labels` 시 `{내용}`만.
  - 그 외 일반표 → 행마다 비어있지 않은 셀을 ` / `로 연결.
- **유지**: `- ` 불릿, `→` 분석줄, `*매크로 종합 …` 줄(볼드만 제거).
- **연속 빈 줄 1줄로 축소**.

### 3. 결과 출력
변환된 평문을 채팅에 그대로 보여주고, 저장 경로를 알려준다. 사용자가 복붙만 하면 되도록.

## 주의 / 판단
- **스크립트 결과를 먼저 확인**하고, 시장 뉴스 행에서 이벤트명이 내용에 안 드러나 모호하면(드문 경우) 그 문단 앞에 이벤트명을 한 번 덧붙여 가독성을 보강할 수 있다(예: `삼성전자 — {내용}`). 그 외엔 스크립트 출력 그대로 사용.
- 표가 깨지거나(헤더 인식 실패) 이상하면 원본 마크다운 표 구조(`| | |`)를 점검 — `convert_community.py`의 표 감지 로직 보강.
- 커뮤니티별로 추가 제약(글자수·줄바꿈)이 있으면 사용자에게 물어 후처리.

- 발행본 학습: `/blog-learn`
- 숏츠 음성 대본: `/blog-shorts`
