# 도담아빠 — 유튜브 숏츠 음성 브리핑 생성 규약

당일 증시 리뷰(`/blog-kr`·`/blog-us` 결과)를 **60초 안팎 음성 브리핑 대본**으로 압축하고, edge-tts로 한국어 음성 MP3(남/여 2종)를 생성한다.

## 데이터 출처 (우선순위)
1. 사용자가 채팅에 일일 리뷰/데이터를 붙여넣었으면 그것 사용
2. 아니면 발행 최종본 `C:\Users\dongh\Desktop\Google Drive 동기화\blog\published\YYYY-MM-DD_{KR|US}_published.md`
3. 그것도 없으면 draft `C:\Users\dongh\Desktop\Google Drive 동기화\blog\draft\YYYY-MM-DD_{KR|US}_draft.md`
4. 그것도 없으면 `python fetch_today_review.py --market {us|korea} [--date YYMMDD]`로 시트 직접 조회

**브리핑에 쓰는 섹션**: 1.시장 뉴스 · 2.매크로 동향 · 3.지수 동향 · 4.섹터(주도주 요약만) · 5.종합 판단. (블로그 글의 핵심을 말로 풀어내는 것이 목적 — 종목 등락률 나열 X)

## 고정 구조 (항상 6단락 + 인트로/아웃트로)

1. **인트로 (1문장)** — `[M월 D일], 미국 증시 브리핑입니다.` (국장이면 "한국 증시 브리핑"). 날짜는 거래 세션 날짜.
2. **핵심 뉴스 (섹션 1)** — 그날 가장 큰 뉴스 1~2건을 *위험선호/시장 흐름*과 엮어 서술. 트리거→결과 인과로. (예: "미·이란 평화협정 임박 소식에 유가가 급락하면서 위험 선호가 되살아났습니다. 스페이스엑스가 사상 최대 규모로 상장해…")
3. **매크로 (섹션 2)** — `매크로는 [엇갈렸습니다 / 우호적이었습니다 / 부담이었습니다].` 뒤에 금리·환율·빅스(VIX)·금 중 핵심만 한 호흡으로.
4. **지수 + 큰 그림 (섹션 3 + 종합 도입)** — `지수는 [4대 지수가 모두 상승]했습니다.` + 종합판단의 큰 그림 한 줄(예: "재랠리를 기대할 강한 상승은 아니지만 최근 급락에서는 벗어나는 모습").
5. **주도주·섹터 (섹션 4 요약)** — 주도주 신호(롱 사인) 강세 + 부진 섹터 + 특이 섹터(금융·전력 등) 압축 2~3문장. 개별 등락률 나열 금지, "강세/부진/약진" 동사 중심.
6. **종합 액션 + 워치 (섹션 5)** — 본인 포트폴리오 스탠스(주도주 중심·현금 비중 방향) + 다음 1~2일 임박 이벤트 워치(다음주 향방).
7. **아웃트로 (1문장)** — `이상입니다.`

## 톤·표현 규약 (TTS 자연스러움이 최우선)

- **존댓말 구어체 브리핑** — "~습니다 / ~입니다 / ~모습입니다". 블로그 일일 리뷰의 단정형(~임/~됨)·압축 호흡은 쓰지 않는다(말로 들어야 하므로).
- **영어·약어는 반드시 한글 음가로 적는다** (edge-tts가 영어 엔진으로 튀어 발음이 어색해지는 것 방지):
  - VIX → **빅스** · long sign → **롱 사인** · SpaceX → **스페이스엑스** · S&P500 → **에스앤피 오백** · NASDAQ → **나스닥** · DOW → **다우** · RUSSELL → **러셀** · ETF → **이티에프** · IPO → **아이피오** · AI → **에이아이** · GOLD → **금 가격**
  - 새 약어가 나오면 같은 원칙으로 한글 음가화.
- **숫자**: `%` → **퍼센트**. 큰 수·환율은 자연스러운 한국어로("달러당 엔이 160까지").
- **길이**: 한국어 **약 320자 ≈ 60초**(rate +8% 기준). 60초 숏츠 목표면 본문 320자 내외로 유지. (유튜브 숏츠는 최대 3분 허용이라 길어져도 업로드는 가능)
- 종목·지표를 빠짐없이 넣지 말 것 — **그날의 한 줄 메시지**(위험선호 회복/조정/패닉 등)를 중심으로 압축.

## 저장 (필수)

- 대본: `C:\Users\dongh\Desktop\Google Drive 동기화\blog\shorts\YYYY-MM-DD_{KR|US}_shorts_script.txt`
  (날짜·시장은 그날 draft/published와 동일하게 — 미장은 세션일+1 KST 규칙)
- 순수 텍스트만(마크다운 헤딩·표·볼드·이스케이프 금지). TTS가 그대로 읽음.

## 음성 생성 (필수)

대본 저장 후 재생성기 실행:
```bash
cd "C:\Users\dongh\Desktop\Google Drive 동기화\blog\shorts"
python make_shorts.py YYYY-MM-DD_{KR|US}_shorts_script.txt
```
- 출력: `..._InJoon.mp3`(남, ko-KR-InJoonNeural) · `..._SunHi.mp3`(여, ko-KR-SunHiNeural)
- 기본 속도 +8%. 빠르게: `--rate +18%`(≈+10%), `--rate +28%`(≈+20%) 식으로. 다른 속도를 별도 파일로 남길 땐 출력명에 `_fast28` 등 접미사.
- 생성 후 길이(초) 확인해 보고. 60초 초과 시 사용자에게 대본 축약 또는 속도 상향 제안.

## 영상화 (참고, 자동 아님)
ffmpeg 미설치라 mp4 자동 합성은 안 됨. MP3를 CapCut·VLLO·유튜브 스튜디오에 올려 배경+자막을 얹는 흐름 권장. ffmpeg 설치 시 음성+배경+자막 mp4 자동 합성 확장 가능.

## 정식 예시 (2026-06-12 미장 — 이 포맷의 표준 샘플)
```
6월 12일, 미국 증시 브리핑입니다.

미국과 이란의 평화협정 임박 소식에 유가가 4퍼센트 가까이 하락하면서 위험 선호 심리가 되살아났습니다. 스페이스엑스가 사상 최대 규모로 상장해 첫날 19퍼센트 뛰며 분위기를 끌어올렸습니다.

매크로는 엇갈렸습니다. 국채 금리는 전 만기 소폭 상승, 달러당 엔이 160까지 올라 부담이 남았습니다. 빅스 지수는 하락하면서 18선 밑으로 떨어졌고, 금 가격도 3.66퍼센트의 강세를 보였습니다.

지수는 4대 지수가 모두 상승했습니다. 재상승 랠리를 기대할 정도의 강한 상승은 아니었지만 최근 중동 충격으로 인한 급락에서는 벗어나는 모습입니다.

주도주인 반도체 장비와 메모리에서 롱 사인이 나타나는 정도의 강세가 두드러졌지만, 빅테크와 소프트웨어 섹터는 부진했습니다. 금리 상승에 금융주의 약진도 볼 수 있었습니다. 전력기기는 최근 하락을 멈추는 상승을 보였지만 상승 추세로의 복원은 아직입니다.

주도주 중심의 포트폴리오를 구성하면서 최근 늘렸던 현금의 비중을 줄여나갈 필요가 있습니다. 미국 시간 일요일 중 종전합의 선언이 다음주 증시의 향방을 가를 것으로 예상됩니다.

이상입니다.
```
